## Instagram Challenge

Creating a instagram website using ruby on rails.

## User Stories

### MVP

```
As a user,
So I can appreciate the different pictures of other users,
I want to be able to see all of them on the home page.

As a user,
So I can see a particular picture,
I want to be able to click on it and see this image only.

As a user,
So I can leave my opinion on pictures that are displayed,
I want to be able to leave a comment on them.

As a user,
So I can see what pictures I have uploaded under my account,
I want to be able to have a page with all of them.

As a user,
So I can use the App,
I want to be able to sign up.

As a user,
So I can use the App,
I want to be able to log in and log out when I want.

```

### Other Stories

```
As a user,
so I can see when I am connected,
I want to be able to see my name on the home page.

As a user,
So other people know who i am,
I want to have a profile picture.

As a user,
So I know who published the picture that are displayed,
I want to be able to see who uploaded the pictures.

```
